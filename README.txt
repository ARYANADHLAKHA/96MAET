SOCIAL NETWORK


This is an interactive Social Network that lets you stay in touch with the most important people in your life. Manage your friends at your convenience, see how your network grows, find mutual contacts when you need them, and all in all, have fun socializing on this engaging application!


This application can be used to display the number of users and friendships. You can load input sequences from a txt and export to a file. You can add and remove friends, view the friends network, see mutual friends, and see the connected users as well.
